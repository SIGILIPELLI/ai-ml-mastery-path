# AI & Machine Learning Mastery Path

A free, structured, module-wise AI & Machine Learning training program — entry
level to master level, with runnable Python code in every module and a
hands-on project at the end of each level. Covers the general-purpose ML
foundation: NumPy/pandas, classical ML with scikit-learn, and neural-network
fundamentals with PyTorch.

**Live site:** https://sigilipelli.github.io/ai-ml-mastery-path/

## Contents

- **Level 1 · Entry** — scientific Python stack, data prep, regression/classification/clustering, evaluation, pipelines, first PyTorch network
- **Level 2 · Intermediate** — gradient boosting/XGBoost, time series, NLP basics, CNNs, model interpretability
- **Level 3 · Advanced** — transformers, MLOps, experiment tracking, distributed training
- **Level 4 · Master** — production ML systems, model serving at scale, monitoring & drift, research-paper reproduction

## Local development

```bash
python3 -m venv .venv
.venv/bin/pip install mkdocs-material
.venv/bin/python -m mkdocs serve
```
